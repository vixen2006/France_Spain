# France & Spain — Acolyte Country Pages

A React + Vite web app with country-specific pages for international students moving to **Spain** and **France**, built as part of the Acolyte platform.

## Routes

| Route | Page |
|-------|------|
| `/` | UK Page |
| `/spain` | Spain Page |
| `/france` | France Page |

## Getting Started

### Prerequisites
- Node.js 18+ installed
- npm or pnpm

### Installation & Running

```bash
# 1. Go into the project directory
cd "Interactive UI Design"

# 2. Install dependencies
npm install

# 3. Start the dev server
npm run dev
```

Then open `http://localhost:5173` in your browser.

### Other Commands

```bash
npm run build    # Build for production
npm run preview  # Preview the production build
```

## Features

- 🇪🇸 **Spain Page** — Madrid hero image, Spanish cities (Madrid, Barcelona, Valencia, Seville, Bilbao, Granada, Salamanca, Málaga), NIE/TIE explainer, N26/CaixaBank/Bizum banking guide, Euro cost of living
- 🇫🇷 **France Page** — Paris hero image, French cities (Paris, Lyon, Toulouse, Bordeaux, Montpellier, Strasbourg), OFII explainer, Euro cost of living
- 🇬🇧 **UK Page** — London hero, UK cities, GBP cost of living
- Visa readiness calculator
- Cost of living comparison table (per country)
- Student testimonials
- Banking & SIM guides per country

## Tech Stack

- React 18
- Vite 6
- Tailwind CSS v4
- React Router DOM v7
- Lucide React icons
- Radix UI components
